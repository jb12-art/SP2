# Auction House – Semester Project 2

A student-only auction web application where users can create listings, place bids using virtual credits, and manage their auction profile.

---

## Project Description

Auction House is an online auction platform built using Vanilla JavaScript and the Noroff Auction House API (v2).

Visitors can browse and search listings without logging in, while registered users with a `@stud.noroff.no` email can participate fully by creating listings and placing bids using virtual credits.

The application focuses on:

- Authentication and API integration
- State handling
- Responsive UI design
- User-centered auction workflows

---

## Features

### All Users (Visitors)

- Browse auction listings
- Search listings
- Filter listings by category
- View listing details
- View bid history

---

### Registered Users

#### Account

- Register using `@stud.noroff.no` email
- Login / Logout
- Access authenticated features

#### Profile

- Edit profile bio
- Update avatar
- Update banner
- View personal listings
- View bidding activity

#### Credits

- View available credits
- Credit balance visible while logged in

#### Listings

- Create listings with:
  - Title
  - Description
  - Deadline date
  - Image gallery (optional)
- Edit own listings
- Delete own listings

#### Bidding

- Place bids on listings
- View live bid history
